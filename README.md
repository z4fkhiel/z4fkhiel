## 📂 Proyectos destacados

### 🌿 [BioComercio — Plataforma de comercio con inteligencia de compra](https://github.com/z4fkhiel/curso-web/tree/main/BioComercio-Oficial)

Tienda virtual por catálogo construida con **Flask + SQLite**. Lo más interesante del proyecto no es el catálogo, sino lo que hay debajo:

- **Motor de descuentos** que analiza el historial de compras del usuario y activa beneficios automáticamente (consumo médico, fidelidad, cross-selling entre categorías)
- **Asistente conversacional** sin LLM externo — detecta intenciones por texto, consulta SQLite y responde con datos reales del historial (desglose por fecha, descuentos próximos a desbloquear)
- **Capa de privacidad por diseño** (`privacy_proxy.py`) — expone tendencias de mercado de forma agregada sin filtrar saldos ni identidad de usuario
- **Bcrypt** para contraseñas, sesiones firmadas, factory pattern, migración automática de esquema
